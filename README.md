# Chartered Institute of Taxation (Kenya) - CITK

A professional, responsive static website built for the **Chartered Institute of Taxation (Kenya)**. The platform is designed to advance the understanding and development of tax policy and law by providing accessible information on the institute's mission, services, and leadership.

## ✨ Features

* **Modern & Responsive Design:** Fully mobile-responsive layout utilizing the Bootstrap 5 grid system.
* **Custom Branding:** Integrated custom CSS to reflect the official CITK brand color (`#115e67`).
* **Sticky Navigation:** An intuitive, sticky navigation bar equipped with dropdowns and anchor links for seamless scrolling.
* **Component-Rich:** Utilizes Bootstrap Cards, Icons, and hover effects for a dynamic user experience.
* **Accessible Legal Links:** Includes a dedicated Privacy Policy tailored to the Data Protection Act 2019 (Kenya).
